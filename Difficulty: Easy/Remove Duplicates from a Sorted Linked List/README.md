Remove all duplicate nodes from the list so that each element appears only once.</span></p>
<p data-start="229" data-end="341"><strong style="font-size: 12pt;">Note:</strong><span style="font-size: 12pt;"> Try not to use extra space. The nodes are arranged in a </span><strong style="font-size: 12pt;">sorted </strong><span style="font-size: 12pt;">way.</span></p>

</span></pre>
